From Nothing to Coder : Practical HTML & CSS

  • Informasi
  • Tata Tertib
  • Benefit

Tipe : Open Class
Pemateri 1 : Mustika
Pemateri 2 : Windu Yoga Nugraha
Moderator : Andika Rizky Ramadhan
Materi : Frontend
Tempat : IndigoHub Makassar

Tujuan:

  1. Memberikan pemahaman dasar kepada peserta mengenai struktur dan sintaks HTML serta prinsip dasar desain menggunakan CSS.
  2. Membekali peserta dengan keterampilan yang praktis dalam membangun halaman web statis secara langsung melalui sesi praktik (live demo).
  3. Mengenalkan penggunaan Git sebagai alat manajemen versi (version control system) dalam proses pengembangan web.
  4. Mengajarkan cara melakukan deployment website menggunakan platform Vercel agar peserta dapat mempublikasikan hasil proyek mereka secara online.



Informasi lebih lanjut Hubungi Indal Awalaikal

TATA TERTIB KEGIATAN

1. Hadir 5 menit sebelum kegiatan di mulai
2. Mengisi daftar hadir yang telah disediakan panitia
3. Mengikuti acara pembukaan
4. Menggunakan pakaian yang sopan dan rapih
5. Peserta dilarang membawa senjata tajam dan obat-obatan terlarang
6. Peserta dilarang menggunakan HP selama materi berlangsung
7. Peserta dilarang ribut
8. Peserta wajib memperhatikan materi yang diberikan
9. Membuang sampah pada tempatnya
10. Hal-hal yang belum ditetapkan akan di tetapkan dikemudian hari
